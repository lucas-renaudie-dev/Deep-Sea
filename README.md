# Deep-Sea

My final project at ATI Université Paris 8, titled "Deep Sea" (in collaboration with 2 other students).
A 10s <b>Blender</b> animation with 3 scenes, each completely modelled, animated, textured and lit from scratch.
Character cloth simulation was done in <b>Marvelous Designer</b>. Post production was done in <b>After Effects</b>, with color grading and sound design.
